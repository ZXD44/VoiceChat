# VoiceChat System for Minecraft Bedrock Edition

ระบบเสียงตามระยะ (Proximity Voice Chat) สำหรับ Minecraft Bedrock Edition

## คุณสมบัติ (Features)
- ระบบเสียงตามระยะ (Proximity Voice Chat)
- เชื่อมต่อกับ Discord (จำเป็นต้องมีระบบ Backend)
- จัดการผ่านไอเทม Discord Remote

## การติดตั้ง (Installation)
1. คัดลอกโฟลเดอร์ `VoiceChat` ไปไว้ใน `development_behavior_packs` หรือ `behavior_packs` ของเซิร์ฟเวอร์
2. ตรวจสอบให้แน่ใจว่าได้เปิดใช้งาน Scripting API ในการตั้งค่าโลก
3. ตรวจสอบ `manifest.json` สำหรับการพึ่งพาโมดูลที่จำเป็น
